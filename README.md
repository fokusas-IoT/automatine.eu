GreitaGears — Minimalus statinis tinklalapis

Trumpas aprašymas

Šis projektas yra labai paprastas statinis tinklalapis skirtas įmonei, kuri remontuoja automatinės pavarų dėžes. Turinys parašytas lietuvių kalba ir gali būti išplėstas priklausomai nuo poreikių.

Failai

- index.html — pagrindinis puslapis
- services.html — paslaugų aprašymas
- about.html — apie įmonę
- contact.html — kontaktų puslapis su paprasta forma
- styles.css — bendri stiliai

Kaip paleisti

1. Atidarykite projekto katalogą.
2. Dukart spustelėkite `index.html` arba atidarykite jį naršyklėje (pvz., File > Open File...).

Kiti žingsniai / pasiūlymai

- Pridėti nuotraukas (darbų galeriją)
- Integruoti veikiantį kontaktinį formą su backend'u arba naudoti formspree/similar
- Pridėti Google Maps embed su tiksliu adresu
- Susikurti paprastą SEO metaduomenų rinkinį ir favicon
